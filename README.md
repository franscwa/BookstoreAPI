## Getting Started

1.  Clone repo
    ```
    git clone https://github.com/franscwa/BookstoreAPI.git
    ```
2.  Create and activate virtual environment
    ```
    python3 -m venv venv
    source venv/bin/activate
    ```
3.  Install packages
    ```
    pip install -r requirements.txt
    ```
4.  Create .env file and set environment variables
    ```
    touch .env
    ```
5.  Start app
    ```
    python3 app.py
    ```
